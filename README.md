# loke.dev

## Development

Run the dev server:

```shellscript
pnpm run dev
```

## Deployment

First, build your app for production:

```sh
pnpm run build
```

Then run the app in production mode:

```sh
pnpm start
```

## Hosting

This website is configured to use Cloudflare workers. You are free to adapt it to any hosting solution.
